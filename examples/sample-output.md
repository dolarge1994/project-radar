What you actually receive and what gets written into your project. (Fictional "Acme Notes" example.)

```
Subject: Acme Notes radar — 1 match(es) — 2026-06-19

Acme Notes daily radar — 2026-06-19
1 new match(es) scored 80+:

ITEM: supabase-mcp-server
SCORE: 88/100
OPPORTUNITY: now — current sprint: wiring AI features to the database
LINK: https://github.com/example/supabase-mcp-server
WHAT IT IS: An MCP server that lets Claude query and manage a Supabase project directly —
schema, rows, and auth — from inside Claude Code.

the hand-written glue between Claude and your database.
RECOMMENDED ACTION: Trial it on a branch this week.

```
